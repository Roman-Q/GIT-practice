ПОЛЕЗНЫЕ ЗАМЕТКИ ПРИ РАБОТЕ С GIT-BASH

Для начала работы нужно подключиться (залогиниться) в профиль GitHub для внесения изменений 

1.В командной строке перейдите в локальный репозиторий и инициализируйте его (папку с файлами, которые нужно запушить "git init") или клонируйте репозиторий с сайта "git clone github://path.git"

2.Добавьте файлы для трека (слежения за изменением) командой "git add" и перечень файлов или папок которые нужно загрузить (через пробел) или убедитесь, что вы находитесь в ветке по умолчанию "git checkout master"

4. Добавьте коммит(сохранение и описание внесенных изменений) командой "git commit -m "release v.1""

5.Получаем доступ к удаленному репозиторию в который хотим запушить файлы командой "git remote add origin https://github.com/kussch/path"

6.Командой "git push -u origin master" пушим необходимые файлы в репозиторий.
